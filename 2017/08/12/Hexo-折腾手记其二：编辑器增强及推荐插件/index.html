推荐插件

这些插件都可以自行选择采用：

博客加密：hexo-blog-encrypt

自动生成RSS：hexo-generator-feed

网页插入PDF：hexo-pdf

自动生成站点地图：hexo-generator-sitemap

THML, CSS, JS, 图片优化：hexo-all-minifier

为文章生成唯一的链接，方便搜索引擎抓取：hexo-abbrlink。不过在使用这个插件时，千万不要在hexo server下直接更改 Markdown 文件名，不然会导致文件空白。
